# Manuscript files

This folder contains the v2.3 journal-ready manuscript files for the ReduLink submission package.

- `ReduLink_journal_ready_v2_3.docx`
- `ReduLink_journal_ready_v2_3.pdf`
- `ReduLink_systems_ready_v2_2.docx` and `.pdf` are retained only as the source baseline from the previous package.

The v2.3 manuscript lengthens the paper, adds independently curated public source-release results, expands related work, methodology, security/privacy, implementation rationale, fairness limits, and limitations, and removes version-specific claim language from the abstract and conclusion.
